Operations that need to be supported by Design~1.
Opcode Operation Microoperations Description
0011 add X MAR ← X # load X into MAR
MBR ← M[MAR] # load value stored at address X into MBR
AC ← AC + MBR # add value in AC with MBR value and store it back into AC 		Adds value in AC with the value at address X and stores the result into AC
0111 halt   Ends the program
0001 load X MAR ← X # load X (address) into MAR
MBR ← M[MAR] # load value stored at address into MBR
AC ← MBR # load value in MBR into AC 		Loads the value from address X into the AC
0010 store X MAR ← X # load address into MAR
MBR ← AC # load AC value into MBR
M[MAR] ← MBR # writes MBR value into the Memory of address indicated by the MAR 		Stores the current value from the AC into address X
1010 clear AC ← 0 Writes 0 into AC
1000 skip C If (C), PC ← (PC) + 1 Skips the next instruction based on C. if (C) is:
0: Skips if AC < 0
2: Skips if AC = 0
4: Skips if AC > 0
1001 jump X PC ← X Jumps to address X
